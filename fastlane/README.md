# Installation

Make sure you have the latest version of the Xcode command line tools installed:

```
xcode-select --install
```

Install _fastlane_ using
```
[sudo] gem install fastlane -NV
```
or alternatively using `brew install fastlane`

# Available Actions
## iOS
### ios lint
```
fastlane ios lint
```
Lint all targets with Swiftlint.
### ios tests
```
fastlane ios tests
```
Run unit tests in iOS sumilator.
### ios archive_for_tests
```
fastlane ios archive_for_tests
```
